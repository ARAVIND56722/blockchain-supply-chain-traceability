🔗 Blockchain-Based Supply Chain Traceability System

A secure and transparent system to track and verify products across the supply chain using blockchain technology and QR-based authentication.

🎯 Problem Statement

Traditional supply chains lack transparency and traceability, making it difficult to verify product authenticity.  
This leads to issues like counterfeit products, data inconsistency, and lack of trust among stakeholders.

💡 Solution

This project provides a blockchain-based system where:

- Manufacturers can register products  
- Each product is assigned a unique QR code  
- Supply chain participants update product status  
- Customers can verify product authenticity  


⚙️ Features

- 🔒 Tamper-proof data storage using blockchain  
- 📦 End-to-end product traceability  
- 📱 QR code-based product verification  
- 👥 Role-based access (Manufacturer, Distributor, Retailer, Customer)  
- 🌐 Transparent and decentralized system  

🛠️ Tech Stack

- **Blockchain:** Ethereum (Sepolia Testnet)  
- **Smart Contracts:** Solidity  
- **Framework:** Hardhat  
- **Frontend:** React.js  
- **Blockchain Interaction:** Ethers.js  
- **Wallet:** MetaMask  
- **QR Code:** JavaScript QR Libraries  
- **Tools:** Node.js, VS Code  

 🔄 How It Works

1. Manufacturer registers product on blockchain  
2. Smart contract stores product details  
3. QR code is generated for the product  
4. Product moves through supply chain  
5. Each stage updates product status  
6. Customer scans QR code to verify authenticity  


📊 Results

- Improved transparency in supply chain  
- Secure and immutable product records  
- Detection of counterfeit products  
- Easy verification for customers  

🔮 Future Scope

- Full supply chain integration (supplier → customer)  
- IoT integration for real-time tracking  
- Mobile app for easier access  
- AI-based fraud detection
